# Diagnosing API issues

## Purpose

To provide some guidance for when there are suspected issues with the Social
Care Case Viewer API.

## Possible issue: API timeouts

The most common issue with the service API that occurs are timeout issues. This
is because it relies on a chain of other APIs i.e. Residents Social Care
Platform API and Mosaic Resident Information API and the combined response times
add up.
